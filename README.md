🚗 Sistema de Estacionamento

Aplicação em Python que simula o gerenciamento de um estacionamento, permitindo registrar entradas e saídas de veículos e calcular automaticamente o valor da permanência.

Funcionalidades

Cadastro de tarifas por tipo de veículo

Registro de entrada e saída

Cálculo automático de permanência

Desconto para pagamento via PIX

Relatório diário e por tipo de veículo

Executar
cd estacionamento
python main.py
📊 Análise de Produtos e Relatórios

Projeto que realiza análise de dados de produtos a partir de um arquivo CSV, permitindo gerar estatísticas e relatórios automatizados.

Funcionalidades

Contagem de produtos por categoria

Percentual de produtos por categoria

Identificação de best sellers

Lista dos 10 produtos mais caros e mais baratos

Geração de relatório HTML automático

Relatório gerado:

relatorio_top_10_best_sellers.html
Executar
cd analise-produtos
python gerar_relatorio_html.py
